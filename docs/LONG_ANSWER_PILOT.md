# Long Answer Pilot (Live-First Validation)

## Goal
Validate implementation quality on a small set before scaling to all chapters.

## Pilot Chapters (live first)
1. Science Ch 4 — Nutrition in Living Organisms
2. History Ch 5 — Foundation of Swaraj
3. Geography Ch 7 — Soils

## Validation Checklist
- Long Answers tab visible in Explanation > Memorize
- 4–8 chapter-end style prompts shown
- Model answers are textbook-like
- Key points are meaningful and chapter-specific
- UI flow works on mobile and desktop

## Rollout Rule
- If pilot approved, proceed batch expansion subject-wise.
- If pilot rejected, patch pattern and retest only pilot set first.
